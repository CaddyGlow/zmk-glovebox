# Glove80 Resource Files

This folder contains configuration files for the Moergo Glove80 keyboard. To be used as resource for development.

## Files Overview

- `factory.json`: The default layout configuration that comes pre-installed on the Glove80 keyboard.
- `factory.keymap`: The generated keymap file from the default configuration that can be built with ZMK tools or flashed directly.
- `glove80_behaviors.json`: Contains behavior definitions specific to the Glove80 keyboard as defined by Moergo in their custom firmware.
- `moergo_glove80_behaviors.json`: Contains behavior definitions and mappings from the my.glove80.com website.
- `zmk_behaviors.json`: Contains the default behavior definitions from the ZMK firmware project.
